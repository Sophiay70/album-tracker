⏺ Two changes, both files:

  &country=us — pins every iTunes request to the US storefront. Without this,
  the API serves results from the user's local storefront, and Jazz/Classical
  catalogues are significantly thinner in many regions — returning 0 results
  rather than returning an error, so no error state ever shows and the section
  just appears blank.

  AbortController — when you switch genres (or change a search field), the
  cleanup function now calls controller.abort(), which cancels the in-flight
  request. The .catch ignores AbortError so it doesn't trigger the error state.
  This prevents a slow Jazz/Classical response from a previous click from
  overwriting whatever genre you've already moved on to.
